Linux Tutorial: Quick Guide

1. Introduction to Linux

Linux is a free and open-source operating system based on Unix. It is widely used for servers, desktops, and embedded systems.

2. Basic Commands

Command Purpose
pwd Shows current directory path
ls Lists files and folders
cd Changes directory
mkdir Creates a new directory
touch Creates a new file
cp Copies files
mv Moves or renames files
rm Deletes files
cat Displays file contents
echo Prints text to the terminal
grep Searches for patterns within files
find Searches for files and directories
chmod Changes file permissions
chown Changes file owner and group
tar Archives files and directories
zip Compresses files into a ZIP archive
unzip Extracts files from a ZIP archive

3. More Linux Commands

Command Purpose
whoami Shows the current username
uname -a Displays system information (kernel name, version)
uptime Shows how long the system has been running
ps Displays information about running processes
kill PID Terminates a process by Process ID (PID)
sudo Runs a command as a superuser (administrator)
wget Downloads files from the internet
scp Copies files between computers over SSH
alias Creates shortcuts for commands
crontab -e Schedules tasks to run automatically
diff file1 file2 Compares two files and shows differences
head Displays the first few lines of a file
tail Displays the last few lines of a file
basename /path/filename Extracts the filename from a path

4. File Permissions

Linux provides an advanced permission system. You can use the chmod command to change the permissions of files. For example:

chmod 755 file.txt

This gives the file owner full permissions, and others can read or execute the file.

5. Package Management

To install or update software on your Linux system, you can use package managers.

Ubuntu/Debian

  • sudo apt update – Update the package list
  • sudo apt install package-name – Install a new package
  • sudo apt upgrade – Upgrade installed packages

Red Hat/CentOS

  • sudo yum install package-name – Install a package using YUM
  • sudo yum update – Update the system
  • sudo yum remove package-name – Remove a package

6. Networking Commands

Command Purpose
ifconfig Displays network interface configuration
ping Checks network connectivity
ssh Connects to a remote system via SSH
netstat Displays network connections, routing tables, and interface statistics
curl Transfers data from or to a server

7. Useful Tips

  • man command – Opens manual for any command.
  • history – Shows previously typed commands.
  • Use the Tab key for auto-completion!
  • clear – Clears the terminal screen.
  • top – Displays system processes and resource usage.
  • df – Displays disk space usage.
  • du – Displays disk usage of files and directories.

8. Conclusion

Learning Linux boosts your technical skills and makes you comfortable with servers, cloud computing, and programming environments.

Mastering basic commands helps you interact easily with the system and automate tasks.

Linux powers most of the world's servers, supercomputers, and cloud platforms, so learning it opens great career opportunities.

Keep practicing different commands daily, and try small projects like creating scripts, managing users, and setting up your own server!

As you progress, explore more advanced topics like shell scripting, cron jobs, system administration, and network security.
